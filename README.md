# 📚 Curso de MySQL - Projetos e Exercícios

Este repositório reúne os exercícios, desafios e projetos desenvolvidos durante meus estudos de **MySQL**, principalmente acompanhando as aulas do **Curso em Vídeo**.

## 🎯 Objetivo

Praticar os principais conceitos de banco de dados relacionais, modelagem e linguagem SQL, construindo uma base sólida para o desenvolvimento de sistemas.

## 📂 Estrutura

Cada pasta contém um projeto, exercício ou banco de dados específico abordando diferentes temas:

* Criação de bancos de dados
* Criação e alteração de tabelas
* Tipos de dados
* Inserção, atualização e exclusão de registros
* Consultas SQL (`SELECT`)
* Filtros e ordenação
* Funções de agregação
* Relacionamentos entre tabelas
* Chaves primárias e estrangeiras
* Boas práticas em bancos de dados

## 🛠️ Tecnologias Utilizadas

* MySQL
* MySQL Workbench
* VS Code
* Git e GitHub

## 🚀 Como Utilizar

1. Clone este repositório:

   ```bash
   git clone https://github.com/GuilhermeSZB/Curso-MySQL.git
   ```

2. Navegue até a pasta desejada.

3. Execute os arquivos `.sql` em seu ambiente MySQL.

## 📖 Aprendizados

Este repositório documenta minha evolução no estudo de bancos de dados e serve como material de consulta para futuros projetos.
